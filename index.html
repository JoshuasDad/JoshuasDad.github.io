HTML & CSS Cheatsheet

Table Tags

Tag Name Description
<table> Table The all encompassing element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The encompassing element containing your actual table data.
<tr> Table Row The element that creates a row.
<td> Table Data The element that contains your data placed in your row element.
<tfoot> Table Foot The set of rows defining the footer in your table.

Table Attributes

Attribute Name Description
<colspan> Column Span Defines how many columns a <td> element will span.
<rowspan> Row Span Defines how many rows a <td> element will span.
